# api-to-elm

Convert the models of a Swagger or GraphQL API to ELM.


## Usage

```
> node index.js api.graphql Api.elm
```

## Development

Requirements:
- [node](https://nodejs.org/en) (tested with v18.7.0)

```
> npm install
> node index.js <input> <output>
```
